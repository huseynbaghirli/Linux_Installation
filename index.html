Linux nədir?

Linux - bu azad və açıq mənbə əməliyyat sistemidir. 1991-ci ildə Linus Torvalds tərəfindən yaradılmış bu sistem, Unix əməliyyat sistemindən ilham alınmışdır və bu gün dünyanın ən populyar əməliyyat sistemlərindən biridir.

user@system:~$
uname -a
Linux hostname 6.1.0-centos10 #1 SMP x86_64 GNU/Linux

Açıq Mənbə

Linux tamamilə pulsuz və açıq mənbədir. İstənilən şəxs kodunu görmək, dəyişdirmək və paylaşmaq hüququna malikdir.

Təhlükəsizlik

Güclü təhlükəsizlik xüsusiyyətləri ilə məşhurdur. Viruslar və zərərli proqramlar üçün daha az həssasdır.

Performans

Yüksək performans və sabitlik təmin edir. Serverlər və superkompüterlər üçün ideal seçimdir.

Çeviklik

Fərdi ehtiyaclara uyğun olaraq fərdiləşdirilə bilər. Müxtəlif distribusiyalar mövcuddur.

Linux-un Tarixi

Linux-un inkişaf tarixi innovasiya və icma əməkdaşlığının parlaq nümunəsidir:

1991

Başlanğıc: Linus Torvalds Finlandiya Helsinkinin Universitetində tələbə iken Linux nüvəsinin ilk versiyasını yazdı.

1992

GPL Lisenziyası: Linux GNU GPL lisenziyası altında buraxıldı və tamamilə azad proqram təminatı oldu.

1993

İlk Distribusiyalar: Slackware və Debian kimi ilk Linux distribusiyaları yarandı.

1994

Red Hat: Red Hat şirkəti yarandı və kommersiya Linux dəstəyi təqdim etməyə başladı.

2000-ci illər

Böyük İnkişaf: Linux server bazarında dominant mövqe qazandı və iri şirkətlər tərəfindən qəbul edildi.

2010-ci illər

Bulud Dövründə: Bulud hesablama və konteyner texnologiyalarının inkişafı ilə Linux daha da populyarlaşdı.

2020-ci illər

Müasir Dövr: Süni intellekt, IoT və Edge Computing sahələrində Linux liderlik edir.

Linux-un Əsas Xüsusiyyətləri

Niyə Linux Yaradıldı?

Linux yaradılmasının əsas səbəbləri:

  • Azadlıq: Proprietary sistemlərdən asılı olmamaq
  • Əlçatanlılıq: Hər kəs üçün pulsuz əməliyyat sistemi
  • İnnovasiya: Açıq inkişaf modeli vasitəsilə sürətli inkişaf
  • Təhsil: Sistemin daxili işlərini öyrənmək imkanı

CentOS 10 haqqında

CentOS (Community Enterprise Operating System) - RHEL (Red Hat Enterprise Linux) əsasında yaradılmış azad və açıq mənbə Linux distribusiyasıdır.

CentOS 10-un Yeni Xüsusiyyətləri

  • Yenilənmiş Kernel: Linux 6.x seriyası ilə daha yaxşı performans
  • Konteyner Dəstəyi: Docker və Podman üçün güclü dəstək
  • Təhlükəsizlik: SELinux və Firewalld gücləndirilmiş versiyaları
  • İdarəetmə: Cockpit web interfeysi ilə asan idarəetmə

Quraşdırma Növləri

Standard Quraşdırma

GUI (Qrafik İstifadəçi İnterfeysi) ilə tam funksional desktop mühiti. Yeni başlayanlar üçün ideal.

  • GNOME və ya KDE desktop mühiti
  • Qrafik proqramlar və alətlər
  • Daha çox sistem resursu tələb edir
  • Desktop istifadəsi üçün uyğun

Minimal Quraşdırma

Yalnız əsas sistem komponentləri. GUI yoxdur, sadəcə terminal (command line) interfeysi.

  • Yalnız terminal (qara ekran) interfeysi
  • Minimum sistem resursu istifadəsi
  • Server və ya xüsusi məqsədlər üçün
  • Əl ilə paket quraşdırma lazımdır

Minimal Quraşdırma Nə Zaman Seçmək Lazımdır?

  • Server İstifadəsi: Web server, database server, mail server kimi
  • Resurs Qənaəti: Az RAM və disk sahəsi olan sistemlər üçün
  • Təhlükəsizlik: Minimum komponent = minimum risk
  • Remote İdarəetmə: SSH ilə uzaqdan idarə olunacaq sistemlər
  • Konteyner Host: Docker və ya Kubernetes node-ları üçün

Windows ilə SSH Əlaqəsi

SSH vasitəsilə Windows-dan CentOS sistemə qoşulun. Bu xüsusilə minimal installation üçün vacibdir.

C:\Users\User>
ssh root@192.168.1.100
The authenticity of host '192.168.1.100' can't be established.
ECDSA key fingerprint is SHA256:xxx...
Are you sure you want to continue connecting (yes/no)? yes
root@192.168.1.100's password:
Last login: Tue Jun 3 15:30:21 2025 from 192.168.1.50
[root@centos ~]# _
[Şəkil yüklənə bilmədi: SSH Connection]
Tools: PuTTY, Windows Terminal, Git Bash
Windows-dan SSH ilə CentOS-a qoşulma

SSH ilə Uzaqdan İdarəetmə

  • PuTTY: Windows üçün populyar SSH klienti
  • Windows Terminal: Built-in SSH dəstəyi
  • Remote Management: Fiziki ekrana ehtiyac yoxdur
  • Secure Connection: Şifrələnmiş əlaqə
  • File Transfer: SCP və SFTP dəstəyi
SSH üçün lazım olan ayarlar: # SSH serveri aktivləşdirin systemctl enable sshd systemctl start sshd # Firewall-da SSH portunu açın firewall-cmd --permanent --add-service=ssh firewall-cmd --reload # SSH status yoxlayın systemctl status sshd

Əlavə Məsləhətlər

  • Yenilənmələr: sudo dnf update əmri ilə sistemi yeniləyin
  • Firewall: Təhlükəsizlik üçün lazım olmayan portları bağlayın
  • Backup: Mühüm məlumatları mütəmadi olaraq ehtiyat nüsxələyin
  • Monitoring: Sistemin performansını izləmək üçün monitoring alətləri quraşdırın

Minimal vs Standard Quraşdırma

Minimal Üstünlükləri

  • Az resurs istifadəsi (512MB RAM)
  • Sürətli quraşdırma (~10 dəqiqə)
  • Yüksək təhlükəsizlik
  • Server işləri üçün ideal
  • SSH remote management

Minimal Məhdudiyyətləri

  • GUI yoxdur (qara ekran)
  • Command line bilikləri lazım
  • Əl ilə paket quraşdırma
  • Desktop istifadəsi mümkün deyil
  • Yeni başlayanlar üçün çətin
